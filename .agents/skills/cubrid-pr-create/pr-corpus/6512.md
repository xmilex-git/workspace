## PR body (verbatim)

<http://jira.cubrid.org/browse/CBRD-25447>

### Purpose

Parallel query 관련 코드 변경이 많아, 초기 개발된 parallel heap scan의 리팩토링을 진행했습니다. 주요 변경 사항은 다음과 같습니다:

1. placement_new에 인자를 전달할 수 있도록 수정
2. porting.h에 likely, unlikely, prefetch 매크로 추가 및 각 C 파일 내 중복 define 제거
3. qfile_connect_list에서 base와 last 모두 dependent_list_id가 존재할 경우 발생하던 dangling pointer 문제 수정
4. pred, rest가 모두 없는 heap scan에서도 parallel heap scan 사용 가능하도록 변경
5. 특정 analytic 함수 수행 시 parallel heap scan mergeable list에서 발생하던 오류를 해결하고, 관련 제약 조건 해제
6. sort merge (mergelist_proc)에서 parallel heap scan 가능하도록 제약 조건 해제
7. parallel heap scan에서 memory mapper 대신 unpack_info_ptr 또는 XASL ID를 활용하여, worker thread에서 XASL을 직접 빌드하도록 변경
8. 가끔 parallel aptr이 포함된 쿼리 실행 시 hang 걸리거나(@ctshim), error code가 잘못 출력되던 현상 해결
