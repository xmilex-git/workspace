## PR body (verbatim)

<http://jira.cubrid.org/browse/CBRD-26355>

### Purpose

Parallel heap scan refactor 과정에서 발생한 자원 미반환 문제를 해결합니다.

### Implementation

group by aggregate 결과의 domain resolve 시, DB_VALUE를 clone하고 clear 하지 않아 발생하던 resource tracker assert()를 해결하였습니다.
또한, 에러 발생 시 임시파일을 반환하지 않고 바로 exit하여 임시파일이 열려있는채로 남아버리는 문제를 해결하였습니다.
parallel subquery execution 시 job execution queue의 pop에서 대기하던 스레드들이 무효화(이미 소멸자가 호출됨)된 pointer를 참조하여 생기는 hang 및 segmentation fault를 해결하였습니다.
