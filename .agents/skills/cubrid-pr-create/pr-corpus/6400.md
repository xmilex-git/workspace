- state: MERGED / base: develop / created: 2025-08-28

## Code changes (summary)

- `qo_rewrite_nonnull_count_select_list()` 신규 함수 추가: SELECT list를 순회하며 count(column) 표현식을 재작성 대상으로 검토
- `qo_rewrite_nonnull_count()` 워커 함수 추가: count 인자가 PT_NAME이고 해당 컬럼이 NOT NULL/PK 제약을 가지면 `PT_COUNT`를 `PT_COUNT_STAR`로 치환
- 대상 spec이 filter predicate가 있는 제약을 가지면 재작성을 취소
- RIGHT OUTER / FULL OUTER / CROSS JOIN이 FROM 절에 있으면 재작성을 건너뜀 (NULL 확장 가능성 때문)
- LEFT OUTER JOIN 대상 spec인 경우도 재작성 대상에서 제외
- `qo_rewrite_queries()`에서 신규 함수를 호출하도록 연결
- `query_rewrite.h`에 함수 선언 추가
