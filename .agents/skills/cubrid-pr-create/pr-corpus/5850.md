## Code changes (summary) (summary based on diffstat)

- Add new `src/query/parallel/px_heap_scan/` module (checker, context, list_stream, manager, memory_mapper, misc, perf_monitor, task) implementing the core parallel heap-scan engine.
- Extend `heap_file.c`/`.h` (+243 lines) with page-range-based heap scan primitives usable from worker threads.
- Add `S_PARALLEL_HEAP_SCAN` scan type and supporting fields to `scan_manager.c`/`.h`, `xasl.h`, `xasl_to_stream.c`, `stream_to_xasl.c`.
- Wire `query_executor.c` and `query_dump.c` to open, run, and trace the new parallel heap scan path.
- Add `PARALLEL` / `NO_PARALLEL_HEAP_SCAN` query hints in `csql_grammar.y`, resolved through `parse_tree.h`/`.c`, `name_resolution.c`, `scanner_support.c`, `view_transform.c`, `xasl_generation.c`.
- Introduce `parallel_heap_scan_threads` system parameter (`system_parameter.c`/`.h`).
- Update root/cs/cubrid/sa `CMakeLists.txt` to build the new sources and add a `.github/CODEOWNERS` entry for the new module.
