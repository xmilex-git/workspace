### Purpose

현재 uncorrelated subquery parallel execution 수행 시, select-list에 JAVASP/PLCSQL이 포함되어 있으면 해당 실행을 포기합니다. 그러나 조건절에 JAVASP/PLCSQL이 포함된 경우에는 실행을 시도하여 hang이 발생하는 문제가 있습니다. 이를 해결하기 위해, 기존에는 select-list만을 검사하던 코드를 수정하여 xasl에 포함된 모든 pred를 순회하며 SP 존재 여부를 검사하도록 변경합니다.
