## Code changes (summary)

- `px_heap_scan.cpp`의 소멸/정리 경로에서 `m_vd`(VAL_DESCR)가 non-null이면 `dbval_ptr`의 각 값을 `pr_clear_value`로 해제하고 `db_private_free`로 해제한 뒤 `nullptr`로 리셋하는 로직 추가.
- task 시작 준비 단계에서 원본 `m_vd`를 그대로 공유하지 않고, 별도의 `new_vd`를 `db_private_alloc`으로 할당하여 `dbval_ptr` 배열까지 깊은 복사(`pr_clone_value`)한 뒤 `m_vd`를 이 복사본으로 교체.

- 목적: 여러 워커가 동일한 vd 인스턴스에 동시 접근하며 발생하던 디버그 빌드의 race condition/assert 실패를 스레드별 복사본 사용으로 제거.
