## PR body (verbatim)

<http://jira.cubrid.org/browse/CBRD-26104>

### Purpose

큐브리드에서는 현재 uncorrelated subquery를 병렬로 실행하지 않습니다. 그러나 이러한 서브쿼리는 서로 독립적으로 실행될 수 있기 때문에 병렬 처리가 가능하며, 병렬로 실행될 경우 다양한 측면에서 성능 향상이 기대됩니다. 이에 따라, 큐브리드에 uncorrelated subquery에 대한 병렬 실행 기능을 추가하고자 합니다.

### Implementation

- xasl->aptr_list 항목들을 병렬 task로 분할하여 큐에 추가하고, 각각 실행 후 결과를 join() 처리
- 여러 thread에서 temp file 및 temp cache에 동시에 접근하므로, 동시성 보장을 위해 mutex 추가
- temp file 및 cache에 대한 mutex 도입에 따라, parallel heap scan의 lock 범위에 대한 리팩토링도 함께 수행
